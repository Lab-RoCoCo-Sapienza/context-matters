Dataset

Our dataset extends a widely used dataset by Armeni et al that represents scenes in 3DSG form. This dataset enable embodied agents to interact with real-world environments. However, the limited number of objects in these 3DSGs poses a challenge for LLM-based agents in generating effective plans for given tasks. We generated an extended dataset that builds on the original by incorporating additional objects into the scenes. This expansion increases the environment's complexity and scope, providing a more rigorous evaluation setting. We release the dataset as a resource for the scientific community to support comparisons across related approaches. Our dataset is subdivided into several splits, each comprising a family of household tasks for which a PDDL domain is provided, as well as a natural language description of the goal and the domain structure. Each task in the split constitutes a variation around the primary theme of the main household task. A subset of the tasks is formulated to be impossible with the available objects, making it necessary to solve a relaxed version of the task, adapted to items available in the environment. The dataset comprises 75 tasks across four scenes (Allensville, Kemblesville, Parole, Shelbiana) which are associated with five distinct domains, described below:
  • Dining Setup: Tasks related to setting the table by placing tableware, with a varying type of meal or event.
  • House Cleaning: The agent is required to collect rubbish and use a mop to clean the floor.
  • PC Assembly: Problems related to gathering PC parts in a room to build a working PC.
  • Laundry: Set of problems requiring to collect dirty clothes (if any), gather them in a washing machine, and operate them by properly using detergent or soap
  • Office Setup: The agent is required to move containers to another room, but it must first empty them

    • Dataset Generation

    To generate the dataset, follow these steps:

    1. Download the original 3D Scene Graphs from the official Gibson website.
    2. Clone the repository containing the dataset creation script and JSON files: 
      git clone https://github.com/EmanueleMusumeci/context-matters-dataset/
    3. Navigate to the repository folder and ensure the downloaded 3D Scene Graphs are placed in the appropriate directory.
    4. Run the dataset creation script: 
      python dataset_creation.py
    5. The script processes the original 3D Scene Graphs to generate the extended dataset, which includes JSON files and PDDL domains. Additionally, an LLM can be utilized during the script to create natural language descriptions of the objects in the scenes by using the parameter --description (python dataset_creation.py --description), default is False. These descriptions provide extra details, such as the color and material of each object, offering valuable context for the planning agent. The PDDL domain are also available in the repository, each corresponding to a specific task split.
